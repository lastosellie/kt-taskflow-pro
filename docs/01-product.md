# 01. Product

> TaskFlow Pro — 팀 업무 관리 풀스택 웹 앱

---

## 타깃 사용자

| 역할 | 설명 | 핵심 니즈 |
|------|------|----------|
| **팀 리더 / 매니저** | 프로젝트를 만들고 팀원에게 태스크를 배분하는 사람 | 전체 진행 현황을 한눈에, 병목 파악 |
| **팀원** | 할당된 태스크를 처리하는 사람 | 내 할 일만 빠르게 확인, 상태 업데이트 |
| **게스트 / 외부 협업자** | 특정 프로젝트에만 초대받은 사람 | 제한된 뷰, 별도 계정 없이 접근 |

**주요 사용 환경**: 데스크톱 브라우저 (우선), 모바일 브라우저 (반응형 지원)

---

## 핵심 기능

### F1. 인증 (Authentication)
- 이메일/비밀번호 회원가입 · 로그인
- 소셜 로그인 (Google OAuth)
- JWT 기반 세션 관리

### F2. 워크스페이스 (Workspace)
- 워크스페이스 생성 · 수정 · 삭제
- 멤버 초대 (이메일) · 권한 관리 (Owner / Admin / Member)
- 워크스페이스 전환

### F3. 프로젝트 (Project)
- 프로젝트 생성 · 수정 · 아카이브
- 프로젝트별 멤버 접근 제어
- 프로젝트 진행률 자동 계산

### F4. 태스크 (Task)
- 태스크 생성 · 수정 · 삭제
- 상태 전환: `Todo → In Progress → Review → Done`
- 담당자 (Assignee) 지정
- 우선순위: `Low / Medium / High / Urgent`
- 마감일 설정 · 기한 초과 알림
- 태그 (Label) 부착

### F5. 보드 뷰 (Board View)
- 칸반 보드 (상태별 컬럼 드래그 앤 드롭)
- 리스트 뷰 전환

### F6. 대시보드 (Dashboard)
- 내 태스크 요약 (오늘 마감 / 기한 초과 / 진행 중)
- 워크스페이스 전체 통계

### F7. 알림 (Notification)
- 인앱 알림 (태스크 할당, 댓글, 마감 임박)
- 이메일 알림 (선택적 구독)

## 비기능 요구사항

| 항목 | 기준 |
|------|------|
| **성능** | 주요 페이지 최초 로드 2초 이내 (LCP 기준) |
| **반응형** | 모바일(360px) ~ 데스크톱(1440px) 전 해상도 정상 동작 |
| **보안** | HTTPS 필수, 비밀번호 bcrypt 해싱, XSS · CSRF 방어 |
| **접근성** | WCAG 2.1 AA 수준 (키보드 탐색, 스크린리더 대응) |
| **확장성** | 워크스페이스당 멤버 200명, 태스크 10,000개 이상 정상 동작 |

## 용어집 (Glossary)

| 용어 | 정의 |
|------|------|
| **Workspace** | 최상위 조직 단위. 회사 또는 팀 하나에 대응. |
| **Project** | 워크스페이스 내 업무 묶음. 특정 목표나 제품에 대응. |
| **Task** | 프로젝트 내 개별 업무 단위. 가장 작은 작업 단위. |
| **Assignee** | 태스크를 책임지는 멤버. 1명만 지정 가능. |
| **Label** | 태스크에 붙이는 자유 태그. 여러 개 가능. |
| **Owner** | 워크스페이스 생성자. 모든 권한 보유. 1명. |
| **Admin** | Owner가 지정한 관리자. 멤버 관리 가능. |
| **Member** | 일반 팀원. 할당된 프로젝트의 태스크만 처리 가능. |
| **Guest** | 특정 프로젝트에만 초대된 외부 협업자. 읽기 중심. |
| **Board** | 태스크를 상태 컬럼으로 시각화한 칸반 뷰. |
